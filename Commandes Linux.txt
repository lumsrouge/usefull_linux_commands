******************************** MANIPULER LES FICHIERS ************************************************************

cat : afficher tout le fichier
	-n	numéro de ligne

less : afficher le fichier page par page
	h affiche l'aide dans less

head : afficher le début du fichier
	-n  [] nombre de lignes désirées

tail : afficher la fin du fichier
	-n  [] nombre de lignes désirées
	-f follow (dynamique)
	-s [] recharger toutes les x secondes

touch [nom_du_fichier] : créer un fichier

mkdir [namedir] : créer un dossier
	-p créer des fichiers intermédiaires (animaux/vertebres/chat)

cp [fichier à copier] (/emplacement)(nom_du_fichier) : copier un fichier
	-R :Copier des dossiers

* : joker *[x] ==> tout ce qui se termine par x
	   [x]* ==> tou ce qui commence par x
		on peut l'utiliser dans cp

mv : déplacer un fichier/renommer

rm : supprimer un fichier
	-i : demander confirmation
	-f : forcer la suppression, quoi qu'il arrive
	-v : dis-moi ce que tu fais, petit cachotier
	-r : supprimer un dossier et son contenu
	JAMAIS: rm -rf /*

ln [fichier1] [fichier2] : crée un lien physique entre fichier1 et fichier2, crée fichier 2
	-s : crée lien physique (raccourci)

******************************************* RECHERCHER DES FICHIERS ************************************************

find [directory] : une recherche approfondie
	-name "x" : cherche un nom. Placer après le dossier.
Option à ajouter à la suite:
	-size +[x]M ou k ou G
	-atime -[nombre de jours]
	-type d : directories only
	-type f : files only
	-printf "[...]"
	-exec [commande] : executer une comamande sur chacun des résultats
	-ok : exec avec configuration

find   -name "*.jpg" -exec chmod 600 {} \; : execute chmod sur chaque fichier {} sera remplacé par chaque fichier

********************************************* EXTRAIRE, TRIER ET FILTRER DES DONNÉES ********************************

grep [option] [texte recherché] [fichier] : guillemets si espace dans le texte
	-i : ne pas tenir compte de la casse
	-n : connaître les numéros des lignes
	-v : inverser la recherche : ignorer un mot
	-E : expressions reégulières

grep -r "Site du Zéro" code/ : recherchera la chaîne «Site du Zéro» dans tous les fichiers du répertoire.
											+ sous dossiers

sort [option] [nom du fichier] [nom du fichier créé si -o] : trier les lignes
	-o : écrire le résultat dans un fichier
	-r : trier en ordre inverse
	-R : trier aléatoirement
	-n : trier des nombres
	
wc [] : compter le nombre de lignes, mots, octets, dans cet ordre
	-l : lignes
	-w : words
	-c : octets
	-m nombre caractères

uniq [option] [file] [new file] : supprimer les doublons
	-c : compter le nombre d'occurrences
	-d : afficher uniquement les lignes présentes en double
	
cut [option] [nomfichier] : couper une partie du fichier
cut -c 2-5 noms.txt : conserve les caractères 2 à 5 de chaque ligne du fichier
	-d : indique quel est le délimiteur dans le fichier
	-f : indique le numéro du ou des champs à couper.
cut -d , -f 1 notes.csv
cut -d , -f 1,3 notes.csv

********************************************** LES FLUX DE REDIRECTION *********************************************

> et >> : rediriger le résultat dans un fichier

2>, 2>> et 2>&1 : rediriger les erreurs

< et << : lire depuis un fichier ou le clavier

| : chaîner les commandes

sudo grep log -Ir /var/log  | cut -d : -f 1  | sort | uniq

1.Elle liste tous les fichiers contenant le mot « log » dans /var/log (-I permettant d'exclure les fichiers binaires).
2.Elle extrait de ce résultat uniquement les noms des fichiers.
3.Elle trie ces noms de fichiers.
4.Elle supprime les doublons.
